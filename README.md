# 🎯 Guess the Number Game

A simple Python CLI game where the user tries to guess a randomly generated number.

## 🚀 Features
- Random number generation
- User input handling
- Feedback (Too high / Too low)
- Score tracking (optional)

## 🛠️ Tech Used
- Python
- random module

## ▶️ How to Run
```bash
python main.py
